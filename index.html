Quote Getter

This is a fun little script that plays to my love of quotes. I keep a spreadsheet of quotes I like, so I thought it would be cool to create a random quote getter from this sheet. The sheet is read in using D3, and then a random index is selected for the quote, author, and source work of the quote. Basic JavaScript is then used to bind the data to specific elements.
